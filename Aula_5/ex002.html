Comments and Annotations

First of all, you need to keep your code clear for those who
read it in the future. Thinking on this, the creators
of JS and some other languages add symbols who basically
says to the compiler "ignore this line"/"ignore this lines".
This symbols are, respectfully: "//" for one line annotations
and the symbols "/*", "*/" to delimit the ignored parts.

Now you already know how to track your knowledge inside the code
we will learn what variables, how they work and how to declare
them.

Variables

Variables are elements that make presence, if not all, in almost
all programming languages. They are responsible for creating
slots in the server/computer memory to store user data.

Naming Your Variables

- An variable name can start with: letter, $ or _
- Can't start with numbers
- It's possible tu use letters and numbers
- It's possible to use accents and symbols
- Can't contain spaces
- Can't be reserved words

Examples: Ex.1: userName / Ex.2: userAge / Ex.3: userGender
